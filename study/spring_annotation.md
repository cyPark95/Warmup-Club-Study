# Spring Annotation

## 스프링 빈을 등록하는 방법

- `@Configuration`
    - 클래스에 붙이는 어노테이션
    - `@Bean`을 사용할 때 함께 사용해 주어야 한다.

- `@Bean`
    - 메서드에 붙는 어노테이션
    - 메서드에서 반환되는 객체를 스프링 빈으로 등록한다.

>- `@Service`, `@Repository`
>  - 개발자가 직접 만든 클래스를 스프링 빈으로 등록할 때 사용한다.
>- `@Configuration` + `@Bean`
>  - 외부 라이브러리, 프레임워크에서 만든 클래스를 등록할 때 사용한다.
